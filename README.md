# Scenes

An agent **skill** for interactive 2D/3D educational scenes and showcases — plus the local tooling to manage and view them.

Scene content is **pretty much pure [Three.js](https://threejs.org/)** in a small portable package. The skill teaches the package shape and the loop; agents write the code.

## Who it’s for

**Teachers and students** — explore a subject or concept with interactive 3D/2D scenes. Creation is **fully via your existing agent of choice** (Claude Code/Cowork, ChatGPT Codex/Cowork, and similar): you talk through the idea; the agent authors and updates scenes. People browse, open, play with the scene and object parameters.

**Model showcase & comparison** — a more organized way to build demo scenes for testing and comparing the **3D capabilities of new AI models** (benchmarks, drop showcases, side‑by‑side libraries).

## What you get

| Piece | Role |
|-------|------|
| **Skill** | How agents author portable scene packages and use the product |
| **CLI** | Init workspace, validate, list, show |
| **Viewer + runtime** | Browse the library, open a scene, parameters, camera, tools |

Ships as **one npm package** when published: skill + CLI + embedded viewer together.
